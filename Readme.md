# Ray Racer
Ray Racer is an Atari 2600 emulator and debugger for macOS.
<img width="600" height="547" alt="RayRacer v0.3" src="https://github.com/user-attachments/assets/5f6dcce9-fbb0-4c6d-99cf-019bf265b3bf" />

## Project status
Ray Racer is under development and offers only partial emulation at present.

It does not emulate

* any controller other than 1 button joystick
* audio
* Atari cartridges with Super Chip
* 3rd party cartridges with bank switching
